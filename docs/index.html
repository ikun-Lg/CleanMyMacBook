CleanMyMacBook

一键禁用键盘、触控板和屏幕，安心擦拭 MacBook

下载 DMG

macOS 13.0+ · ARM / Intel 通用

功能特性

⌨️

禁用键盘

拦截所有按键输入，放心擦拭键盘不留误触

🖱️

禁用触控板

拦截点击、移动与滚动，清洁触控板无干扰

🌑

纯黑屏幕

全屏黑色遮罩，灰尘污渍一目了然

⏎

快速退出

按住 Esc 键 2 秒即可退出清洁模式

🔒

安全透明

不收集任何数据，所有操作本地完成

🪶

轻量原生

SwiftUI 构建，菜单栏常驻，几乎零资源占用

使用步骤

1

下载并安装

下载 DMG 文件，将 CleanMyMacBook 拖入 Applications 文件夹

2

选择清洁选项

点击菜单栏图标，勾选要禁用的功能：键盘、触控板、黑屏

3

开始清洁

点击「开始清洁」，首次使用会请求辅助功能权限

4

退出清洁模式

按住 Esc 键 2 秒，所有功能恢复正常

安装方式

选择适合你的方式开始使用

下载

DMG → Applications

源码构建

./build_dmg.sh

系统要求：macOS 13.0 (Ventura) 或更高 · Apple Silicon / Intel 通用
